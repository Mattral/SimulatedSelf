face-api.js (vladmandic fork) model weights
-------------------------------------------

The following files MUST be present in this directory for the
worker-backed emotion pipeline (src/workers/vision.worker.ts) to load:

  - tiny_face_detector_model-weights_manifest.json
  - tiny_face_detector_model.bin
  - face_expression_model-weights_manifest.json
  - face_expression_model.bin

Source:
  https://github.com/vladmandic/face-api/tree/master/model

The CI workflow (.github/workflows/ci.yml) verifies all four files
exist on every PR and fails the build if any are missing.

The worker also performs a runtime preflight HEAD/GET against each
manifest URL on init and emits a clear `MODEL_LOAD_FAILED` error
if a file 404s in production.
